# 🎓 Student Internship Tracker

A desktop-based Java application that helps students manage internship opportunities, applications, and skills in one place.

## Features

- Student Login
- Dashboard
- Internship Management
- Application Tracking
- Skills Management
- Internship Recommendations
- MySQL Database Integration
- Modern Java Swing UI

## Tech Stack

- Java
- Java Swing
- JDBC
- MySQL
- MySQL Connector/J

## Requirements

- Java 17 or later
- MySQL Server
- MySQL Connector/J

## Setup

1. Create the MySQL database.
2. Run the SQL scripts from the `sql` folder.
3. Update database credentials in `DBConnection.java`.
4. Add `mysql-connector-j-8.0.31.jar` to the project.
5. Compile and run `MainApp.java`.
